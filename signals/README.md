# signals/

Signal detector definitions organized by category. See [docs/signal-catalog.md](../docs/signal-catalog.md) for the full catalog.

## Categories

- `health/` -- slowTest, flakyTest, skippedTest, deadTest, unstableSuite
- `quality/` -- weakAssertion, mockHeavyTest, untestedExport, coverageBlindSpot, etc.
- `migration/` -- frameworkMigration, migrationBlocker, deprecatedTestPattern, etc.
- `governance/` -- policyViolation, legacyFrameworkUsage, skippedTestsInCI, runtimeBudgetExceeded

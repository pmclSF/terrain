# Paid Product

## Strategy

The open-source Hamlet must be indispensable to developers.

The paid Hamlet must be indispensable to organizations.

## Open Source includes

- local repository analysis
- signal generation
- CLI
- local JSON snapshots
- extension
- local policy checks
- migration readiness
- CI-friendly output

## Paid product includes

### Historical intelligence
- trends across snapshots
- risk movement over time
- quality drift
- runtime trends

### Cross-repo visibility
- org rollups
- service/repo comparisons
- team-level aggregation

### Benchmarking
- compare with similar organizations
- compare by size / language / framework mix
- percentile comparisons

### Governance
- centralized policy
- org-level risk surfaces
- risk budgets
- hosted dashboards
- alerting

## Important rule

Do not paywall the core signals.

Monetize:
- aggregation
- history
- benchmarking
- centralization
- governance
